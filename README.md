# sudoku_solver
Solve sudoku puzzle

## Usage
./solver data/game0.txt

## Input format
File should have exactly 81 character except line brakes. Numbers are given values, spaces are blanks. See data folder for examples.
```ruby
 35  1  7
94 7 8 5 
2 76 4   
42     76
   1 2   
81     29
   3 97 5
 5 2 7 64
7  5  21 
```
